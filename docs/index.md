# Qu'est ce que Windoc ?

Une bibliotèque de documentation sur la voile.

Elle a vocation à servir de lieux de rangement et synthèse de mes connaissances pour usage ulterieure.  
Je la publie sur le net comme un exercice d'utilisation de [MkDocs](https://www.mkdocs.org/), [Git](https://fr.wikipedia.org/wiki/Git) et d'autres outils. Ainsi que dans l'idée quelle puisse être utile à d'autres.
